Sobre mim

Olá! Meu nome é Felipe Silva e sou estudante do curso de Desenvolvimento Full Stack Java da EBAC (Escola Britânica de Artes Criativas e Tecnologia). Durante minha trajetória acadêmica, venho adquirindo conhecimentos importantes sobre as diversas áreas da programação, com foco nas tecnologias que são essenciais para o desenvolvimento de sistemas web. Atualmente, estou me aprofundando no estudo do Java, mas também tenho dedicado tempo ao aprendizado de ferramentas que contribuem para a construção de interfaces de usuário e sites dinâmicos.

Recentemente, comecei a explorar o desenvolvimento frontend, aprendendo HTML e CSS, e adquirindo habilidades para criar páginas web estruturadas e visuais. Essas tecnologias têm sido a base do meu aprendizado em design de interfaces, o que me permite criar experiências mais agradáveis e funcionais para os usuários. Estou empolgado com os desafios que a área de desenvolvimento oferece e continuo buscando oportunidades para expandir meus conhecimentos e melhorar minha atuação como desenvolvedor. Este portfólio é um reflexo desse processo contínuo de aprendizado e evolução.
